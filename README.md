SwinUNETR
![Paper](https://img.shields.io/badge/Paper-arXiv:2201.01266-b31b1b?style=flat-square&logo=arxiv&logoColor=white)
![Conference](https://img.shields.io/badge/Conference-MICCAI%202022-0066cc?style=flat-square)
Swin UNETR is a state-of-the-art architecture for 3D medical image segmentation, combining the power of Swin Transformers with the classic UNETR design. This implementation is based on the paper:
> *"Swin UNETR: Swin Transformers for Semantic Segmentation of Brain Tumors in MRI Images"*  
> Hatamizadeh et al., MICCAI 2022
The model leverages hierarchical Swin Transformers as the encoder to capture both local and global contextual information, making it particularly effective for challenging medical imaging tasks like brain tumor segmentation (BraTS).
